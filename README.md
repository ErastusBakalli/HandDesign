# HandDesign
Hand gesture controlled 3D cube using Python, OpenCV, MediaPipe and OpenGL

## Installation
git clone https://github.com/ErastusBakalli/HandDesign.git
cd HandDesign
pip install -r requirements.txt
python handdesign.py

## Requirements
- Webcam
- Python 3.x

## Features
- Move cube with hand
- Rotate cube using wrist
- Resize cube using fingers
